# PromptLab 🧪

PromptLab is an AI-powered prompt optimization and evaluation tool built with Streamlit and OpenAI.

## Features

- Optimize prompts into 3 improved versions
- Run A/B/C prompt tests
- Score outputs for clarity, completeness, professionalism, and overall quality
- Highlight the best version
- Export results as JSON or CSV
- Track prompt history during the session

## Built With

- Python
- Streamlit
- OpenAI API
- python-dotenv
